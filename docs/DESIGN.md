# VoiceDraw 设计文档

> 七牛云 × XEngineer 暑期实训营 · 题目二：AI 语音绘图工具
> 一款**纯语音控制**的 AI 绘图工具：不用鼠标键盘，只靠说话完成"精准矢量绘图 + AI 文生图 + 局部重绘 + 流程图"。

---

## 1. 产品定位

把工具从"基础画板"提升为**全语音驱动的视觉工作台**：
- **效率模式（本地）**——口述即出图的精准矢量绘图（图形/流程图/线框图），响应 <100ms、零成本；
- **创意模式（云端）**——口述画面即生成 AI 图，并可继续用语音**局部重绘**调整。

二者同处一块画布，可叠加：AI 位图作背景/元素，矢量图形语音叠加标注，实现"氛围图 + 标注"的混合创作。

## 4. 成本控制技巧（计划 / 实际采用）

> 题目要求：想到了哪些控制运营成本的技巧、实际采用了哪些。端云协同的核心是"能本地就别上云、上云就别重复"。

**实际采用：**
1. **本地优先**：高频原子指令（画图形/改色/变换/撤销…）全部走本地规则引擎，**0 次大模型 / 0 次图像调用**——省掉了绝大多数请求。
2. **NLU 用便宜模型**：复杂语句兜底用 `deepseek-v3`，且**仅在本地规则未命中时**才调用。
3. **生图缓存**：相同 `prompt + model` 命中后端缓存，**不重复生成**，仪表盘显示"缓存命中"次数。
4. **模型分级**：文生图用质量更好的 Pro，局部重绘用更快更省的 flash；并保留 `fast` 降级开关。
5. **防抖**：语音中间结果（interim）不触发任何云端调用，仅最终结果触发，避免边说边烧钱。
6. **统一计量 + 可视化**：后端集中计量调用次数与估算成本，前端**实时成本仪表盘**，把端云成本变成可演示、可决策的数字。

**想到但未采用（及原因）：**
- 本地轻量扩散模型替代云端生图——模型太重，一天内集成不现实，且本地推理质量/速度不稳；
- 近似语义缓存（相似 prompt 复用旧图）——收益不稳定、易出错图，时间不足未做。

## 5. 技术架构

```
[麦克风] → Web Speech ASR(zh-CN 连续识别) → 文本
                         │
                    指令路由(router)
        ┌────────────────┼─────────────────────────┐
   本地规则引擎       LLM 兜底(NLU)            创意模式
   (效率模式,<100ms)  复杂语句/流程图/线框图    文生图 / 局部重绘
        │                │                         │
        └──────→ 画布状态(zustand + 撤销/重做) ←─────┘
                         │
              SVG 渲染层(矢量 + AI 位图 + 风格滤镜)
                         │
                  Web Speech TTS 语音反馈

后端代理(Express, 持有七牛密钥) → 七牛 AI 大模型推理 https://api.qnaigc.com/v1
  · /api/nlu      → deepseek-v3 (chat)
  · /api/image/generate → gemini-3.0-pro-image-preview (chat/completions 多模态)
  · /api/image/edit     → gemini-2.5-flash-image (chat/completions 多模态, 免蒙版)
  · 生图缓存 + 成本计量
```

**技术栈**：前端 Vite + React + TypeScript，SVG 场景图，zustand 状态管理，zod 校验；流程图 dagre 自动布局；语音 Web Speech API（ASR + TTS）；后端 Node + Express 代理；云端能力全部走七牛 OpenAI 兼容接口。

**关键工程决策见 [DECISIONS.md](DECISIONS.md)**（含 API 选型实测对比与踩坑记录，用于汇报）。

## 6. 创新点

1. **双引擎混合画布**：同一画布上既能用语音"搭积木"画精准线框/流程图（结构化），又能口述生成 AI 氛围图（非结构化），且可叠加——市面少见的结构化+非结构化混合创作。
2. **语音 → 逻辑 → 流程图**：不是"画矩形再画箭头"，而是口述业务逻辑，系统自动解析开始/操作/判断/结束节点并排版连线，接近"自然语言编程式绘图"。
3. **生成即模板、可说可改**：AI 图生成后即成为可语音微调的模板，免蒙版局部重绘，延续"可迭代、可修正"的产品思维。
4. **实时成本仪表盘**：把"端云协同成本控制"这一评分点直接**可视化**，演示时可量化展示本地优先+缓存省了多少。

## 7. 演示脚本（约 5 分钟，纯语音）

1. **效率模式（1.5 min）**："画一个蓝色矩形" → "画一个红色的圆" → "大一点" → "向右移" → "切换为手绘风格"。展示低延迟、确定性、零成本。
2. **流程图（1 min）**："画一个登录流程图：开始，输入账号密码，验证是否正确，正确进入首页，错误提示重试。" 自动出图、判断分支标注。
3. **创意模式·文生图（1 min）**："切换到创意模式" → "画一幅雨夜的东京小巷，霓虹灯倒映在湿路面，赛博朋克风格"。加载动画 + 语音反馈，几秒出图。
4. **局部重绘（1 min）**："把整体氛围改成暖橙色调"。展示免蒙版、构图保留的指令式编辑。
5. **收尾（0.5 min）**：展示成本仪表盘——"本地指令 0 成本，云端按需调用并缓存"。说明被主动砍掉的像素级自由涂抹是基于体验红线的产品决策。
