# VoiceDraw · 纯语音 AI 绘图工具

> 七牛云 × XEngineer 暑期实训营 · 题目二「AI 语音绘图工具」
> 不用鼠标键盘，全程用语音完成「精确矢量绘图 + AI 创意生图」的创作工作台。

暖萌画家小熊「小绘」作为语音搭档，用 待机 / 说话 / 绘画 / 画图成功 四段动画在合适时机陪伴反馈。

## ✨ 功能亮点

- **双引擎混合画布**
  - **效率模式（矢量）**：图形 / 颜色 / 方位绘制 / 移动缩放旋转 / 增删改 / 撤销重做 / 风格滤镜，本地规则驱动 `<100ms`、零成本、可离线。
  - **创意模式（AI）**：文生图 + 语音局部重绘 + 作品库，云端大模型生成与编辑。
- **一句话出图表**：流程图 / 组织架构图 / 思维导图 / 树状图 / 线框图，LLM 抽结构 + dagre 自动排版；支持节点就地改名/删除。
- **对话式确认**：生图/改图前先确认、可语音纠错；改图时用视觉模型 `qwen-vl` 真看图后再发请求 —— 一招治「限流 / 乱断句 / 不准」。
- **端云协同成本控制**：本地优先、生图缓存、确认门降调用、429 退避重试，并有实时成本仪表盘。
- **纯语音**：阿里 DashScope 流式 ASR 实时字幕 + 浏览器 TTS 语音反馈。

详见 [设计文档](docs/设计文档.md)（含「计划支持 / 最终实现 / 未完成原因」全表）。

## 🧱 技术栈

| 层 | 技术 |
|---|---|
| 前端 | Vite · React · TypeScript · zustand · SVG · @dagrejs/dagre · Geist · Phosphor |
| 后端 | Node · Express 代理（持密钥 / 计量 / 缓存 / 限流重试） |
| 语音 | 阿里云 DashScope `paraformer-realtime-v2`（流式 ASR）· 浏览器 SpeechSynthesis（TTS） |
| 大模型 | 七牛云 AI 大模型推理（OpenAI 兼容）：对话理解 · `gemini` 图像（文生图/局部重绘）· `qwen-vl`（视觉看图确认） |

## 🚀 本地运行

```bash
# 1. 安装依赖
npm install

# 2. 配置密钥
cp .env.example .env
#   然后编辑 .env，填入你的 QINIU_API_KEY 与 DASHSCOPE_API_KEY

# 3. 启动（前端 5173 + 后端代理 8787 一起跑）
npm run dev
```

用 **Chrome** 打开 http://localhost:5173 ，首次点「开始语音」时允许麦克风权限。
缺少某个 key 时，对应云端功能自动隐藏，本地绘图照常可用。
